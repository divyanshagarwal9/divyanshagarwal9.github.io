    VGRNet : A View Invariant Gait Recognition Network

    Learning Style Compatibility for Furniture

    Collected a large scale Furniture dataset with images as well as metadata.  Investigated how Siamese networks can be used efficiently for assessing the style compatibility between images of furniture items. Also used a joint image-text embedding method that allows for the querying of stylistically compatible furniture items, along with additional attribute constraints based on text.
